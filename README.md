# Nadya Yashchuk Portfolio

One-page React portfolio landing page for a Product Manager / Project Lead.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Edit content

- Main page content lives in `src/main.jsx`.
- Visual styling lives in `src/styles.css`.
- The profile photo is `1701113988652.jpeg`.
- Replace `hello@example.com` in the contact section with the preferred email address.

## Build

```bash
npm run build
```
